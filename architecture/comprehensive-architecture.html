üì¶ processBatch(WALEntryBatch batch)

BatchProcessResult processBatch(WALEntryBatch batch)

Specific Task:

Execute an atomic multi-entity transaction with full ACID guarantees.

  • 1 Write entire batch to WAL for durability (walProducer.append)
  • 2 Start database transaction (connection.setAutoCommit(false))
  • 3 Process each entry: convert WAL data to entity
  • 4 Apply each entry to database within transaction
  • 5 Commit database transaction (connection.commit)
  • 6 Update unified cache storage (storage.put for each entity)
  • 7 Return success result or rollback on failure

üîÑ replayWAL()

void replayWAL()

Specific Task:

Restore cache state by replaying all WAL entries from persistent storage.

  • 1 Read batches from WAL consumer (walConsumer.readNextBatch)
  • 2 Convert each WAL entry to entity object
  • 3 Apply to cache storage without database writes
  • 4 Continue until no more batches available
  • 5 Mark replay as complete
